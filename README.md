This add-on provides a clear and organized overview of all CS2 Charms in your inventory, displaying each item’s pattern number and details. Users can easily sort and filter their charms by pattern, name, or value range, and choose to show only special numbers or trade-locked items. Compatible with other addons.

The interface includes:
- Total item count with separate stats for tradable, locked, and special charms
- A detailed list showing charm names (e.g. Baby’s AK, Lil’ Crass, Big Kev)
- Pattern numbers (e.g. 99999, 99997, 99996) with Star symbols for special ones
- A convenient Export button in the top-right corner
- A clean, modern layout with intuitive filter, sort, and search options

Perfect for CS2 collectors, traders, and skin enthusiasts who want to manage their charms efficiently and track rare pattern numbers with ease.
